# Pizza Sales Data Analysis & Visualization

## 📌 Project Overview
This project focuses on analyzing pizza sales data to generate actionable business insights using SQL and Excel. The goal was to transform raw transactional data into meaningful KPIs and interactive dashboards to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze key business performance indicators (KPIs)
- Identify sales trends and customer behavior
- Evaluate product performance across categories and sizes
- Build an interactive and visually insightful dashboard

---

## 🧮 Key Performance Indicators (KPIs)
- **Total Revenue** – Sum of total sales value  
- **Average Order Value (AOV)** – Revenue per order  
- **Total Pizzas Sold** – Total quantity sold  
- **Total Orders** – Number of orders placed  
- **Average Pizzas per Order** – Quantity per order  

---

## 📊 Data Visualizations
The dashboard includes the following charts:

- **Daily Trend of Total Orders** (Bar Chart)  
- **Hourly Trend of Orders** (Line Chart)  
- **Sales by Pizza Category** (Pie Chart)  
- **Sales by Pizza Size** (Pie Chart)  
- **Total Pizzas Sold by Category** (Funnel Chart)  
- **Top 5 Best-Selling Pizzas** (Bar Chart)  
- **Bottom 5 Worst-Selling Pizzas** (Bar Chart)  

---

## 🛠️ Tools & Technologies
- **SQL Server Management Studio** – Data extraction and querying  
- **Microsoft Excel** – Data cleaning, analysis, and dashboard creation  

---

## 🔍 Key Insights
- Identified peak order hours and high-demand days  
- Determined most popular pizza categories and sizes  
- Highlighted top-performing and underperforming products  
- Provided insights for improving sales and inventory planning
